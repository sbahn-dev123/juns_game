# v1.0.3 업데이트 내역 (2026-02-22)

안녕하세요, 용사님!

게임의 재미와 편의성을 향상시키기 위한 몇 가지 업데이트가 적용되었습니다.

## 1. 밸런스 조정

### 모든 보스 몬스터 상향
- 모든 중간 보스와 메인 보스의 체력(HP)이 약 15%, 공격력(ATK)이 약 10% 증가했습니다.
- 이제 보스전이 더욱 긴장감 넘치고 도전적으로 변경되었습니다. 강력해진 보스들을 상대로 여러분의 한계에 도전해보세요!

## 2. 편의성 개선

### 물약 정보 표시 개선
- '물약 사용' 창에서 각 물약의 이름 아래에 상세 효과(회복량, 버프 종류, 지속 시간 등)가 표시되도록 UI가 개선되었습니다.
- 이제 어떤 물약을 사용할지 더 쉽고 직관적으로 판단할 수 있습니다.
