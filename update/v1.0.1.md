# v1.0.1 업데이트 내역

이번 업데이트는 사용자 경험(UX) 개선, 버그 수정, 그리고 게임 밸런스 조정에 중점을 두었습니다.

## ✨ 주요 개선 사항

### 1. UI/UX 개선
- **모바일 스크롤 최적화**: 모바일 환경에서 인벤토리, 상점, 물약 창 등 모든 모달 창의 스크롤 동작을 개선하여 잘림 현상 없이 모든 콘텐츠를 볼 수 있도록 수정했습니다.
- **스크롤 방식 통일**: '이중 스크롤' 문제를 해결하고, 각 창의 스크롤 방식을 더 직관적으로 변경하여 사용성을 높였습니다.
- **반응형 레이아웃 강화**: 다양한 기기 화면 크기에 맞춰 게임 UI가 깨지지 않고 꽉 차게 표시되도록 전체적인 레이아웃 구조를 개선했습니다.
- **UI 재구성**:
    - 메인 컨트롤의 '인벤토리' 버튼을 '장비', '전리품', '스탯'으로 분리하여 각 기능에 더 빠르게 접근할 수 있도록 변경했습니다.
    - 물약 창의 UI를 종류별로 분류된 섹션으로 다시 구성하여 원하는 물약을 더 쉽게 찾을 수 있도록 했습니다.

### 2. 기능 추가
- **게임 오버 화면 개선**: 게임 오버 시, 단순히 알림 창 대신 최종 점수와 함께 '새 게임', '메인으로' 버튼이 포함된 전용 모달 창이 표시되도록 개선했습니다.

### 3. 버그 수정 및 밸런스 조정
- **'마력' 스탯 버그 수정**: '마력(mag)' 스탯의 실제 기능(스킬 피해량, MP 회복량 증가)이 설명과 일치하지 않던 버그를 수정하고, 관련 UI와 게임 설명서를 모두 업데이트했습니다.
- **'마력' 스탯 밸런스 조정**: 게임 밸런스를 위해 '마력' 스탯의 스킬 추가 피해량과 MP 회복량 계수를 하향 조정했습니다.
- **스탯 미리보기 오류 수정**: 스탯 분배 창에서 스탯을 찍었을 때, 실제 적용될 값과 다르게 표시되던 미리보기 오류를 수정했습니다.

### 4. 기타
- **코드 가독성 향상**: 프로젝트의 모든 코드 파일에 상세한 주석을 추가하여 유지보수성과 가독성을 크게 향상시켰습니다.
